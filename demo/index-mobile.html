CalenStyle

Visual and Functional Demonstration of CalenStyle for Mobile Devices

  • While viewing demo on Firefox if you can not see font icons, it is because of folder structure of CalenStyle plugin distribution folder and Strict Origin Policy of Firefox. While implementing CalenStyle when you will keep the folder structure simple, you will be able to see icon fonts. Check CalenStyle Demo.
  • You will not be able to see output of jQuery Mobile example on latest versions of Chrome, since Chrome gives a Security Error. Please check it on some other browser.
